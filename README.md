## Come funziona il punteggio

| Risultato | Punti Vincitore | Punti Perdente |
|-----------|-----------------|----------------|
| 2 - 0     | 3               | 0              |
| 2 - 1     | 2               | 1              |

I mazzi entrano nella classifica "ufficiale" dopo **3 partite giocate**. Prima di allora sono marcati come `prov`.

## Come registrare una partita

1. Apri il sito sul telefono o sul pc
2. Vai sulla scheda **+ Partita**
3. Compila il modulo e premi **Registra Partita**
4. Premi **⬇ Scarica dati.json** (o **⧉ Copia**)
5. Apri il repository GitHub, clicca sul file `dati.json`, premi la matitina ✏️, incolla / sostituisci il contenuto e fai **Commit changes**
6. Aspetta 30 secondi e ricarica il sito: la classifica è aggiornata!
